@ 트리거(TRIGGER) : 연쇄반응
 -> 데이터베이스가 미리 정해놓은 조건을 만족하거나 어떤 동작이 수행되면 자동적으로 수행되는 행동
 -> 테이블이나 뷰가 INSERT, UPDATE, DELETE 등의 DML 문에 의해 데이터가 입력, 수정, 삭제될 경우 자동으로 실행
 EX) 회원탈퇴가 발생하면 기존 회원 테이블에서 삭제하고 해당 정보를 탈퇴 테이블로 이동
	-- 일정 기간이 지나면 탈퇴 페이지에서도 삭제

@@ 트리거 실행 시점
 -> 이벤트 전(BEFORE), 이벤트 후(AFTER)로 지정
@@ 이벤트 : INSERT, DELETE, UPDATE
@@ FOR EACH ROW 생략되면 문장 레벨 트리거
		   정의되면 행 레벨 트리거
	-> 문장레벨트리거 : DML에 의해 한 번에 여러 개의 변화가 발생하더라도 트리거는 한 번 수행
	-> 행레벨트리거 : DML에 의해서 한 번에 5개의 행이 변경되면 트리거가 5번 수행




@ 바인드 변수
:OLD - 기존 데이터(DELETE, UPDATE)
	:OLE.컬럼명 -> SQL 반영 전의 컬럼 데이터
:NEW - 새로 입력된 데이터(INSERT, UPDATE)
	:NEW.컬럼명 -> SQL 반영 후의 컬럼 데이터
